## ⚛️ Quantum Computing Lab

<div align="center">

| Experiment | Key Finding | Metric |
|:-----------|:------------|:-------|
| **[Quantum k-NN](https://github.com/EmaadAkhter/Quantum-K-Nearest-Neighbor)** | Classical wins in NISQ era — ~70s quantum vs milliseconds classical, 10–15% accuracy gap | Classical **93%** · Quantum **76%** |
| **[Quantum Random Forest](https://github.com/EmaadAkhter/Random-Forest-Using-QRNG)** | QRNG bootstrap yields better CV stability despite 500× runtime overhead | Best CV score: **0.9683** |
| **[Quantum Polynomial Regression](https://github.com/EmaadAkhter/Quantum-Polynomial-Regression)** | VQCs match classical regression on polynomials; 4 layers is the sweet spot | R² = **0.982**, MSE = 0.115 |
| **[Quantum CNN](https://github.com/EmaadAkhter/Quantum-Convolutional-Model)** | 4-qubit sliding-window quantum convolution on MNIST; hybrid classical head | QCNN on MNIST |

</div>
